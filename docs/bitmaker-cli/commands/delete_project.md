# bitmaker delete project

## Description

Delete a project in Bitmaker Cloud.

## Usage

```bash
$ bitmaker delete project PID
```

## Options

|Option|Description|
| ---- | --------- |
|PID (Required)|The project's id.|

## Examples

```bash
$ bitmaker delete project 7ba2ceb9-5675-4f27-9d7c-dad54afd40c6
```
