# estela list project

## Description

List projects the active user has access to.

## Usage

```bash
$ estela list project
```

## Examples

```bash
$ estela list project
NAME             PID
quotes           7cf2fda9-5675-4f27-9d8c-faf54ead40c5
proj_test        80ea825d-a13f-48ac-87f0-0a905acfb1d7
book             afcda781-6a70-4022-9b68-b0a3a25f732f
```
